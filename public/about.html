About Application

AttendEase

Version 1.0.0

AttendEase is a comprehensive platform designed to streamline student attendance tracking for educational institutions. The application provides dedicated dashboards for administrators to manage teacher accounts, and for teachers to easily mark daily attendance, manage their student lists, and generate Excel reports.

Core Features:

  • Role-based access control (Admin & Teacher)
  • Bulk student upload via Excel (.xlsx)
  • Daily attendance marking with automated date tracking
  • One-click Excel report generation

Technology Stack:

Node.js & Express MongoDB HTML5 & CSS3 Vanilla JS
